# Workspace Instructions

This is a collaborative project between an AI Agent and a User.

## Core Collaborative Principles

- **Respect User Updates:** Always respect and preserve changes manually updated by the user. 
- **No Regressions:** Do not revert code, parameters, or configurations back to previous AI-generated versions if the user has since modified them.
- **Verify Before Modifying:** Always read the current state of a file before applying any changes or overwrites to ensure user-made updates are not lost.
- **Technical Context:** For this Remotion project, adhere to the established modular scene structure and best practices identified during the development lifecycle.
