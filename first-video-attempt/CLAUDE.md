# Project Instructions

## Collaboration Guidelines

This is a collaborative Remotion project between the user and AI assistant.

**Always respect user changes:**
- Before modifying any file, read it first to check for manual changes made by the user
- Keep all user modifications unless they directly conflict with the current task
- Do not overwrite user changes based on your previous memory or earlier versions
- When the user asks for a new task, preserve their edits from previous work
- Only modify user changes when absolutely necessary for the task at hand

This ensures productive collaboration and prevents loss of user work.
